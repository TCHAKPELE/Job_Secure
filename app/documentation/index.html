Guide d'installation du projet

Clonage du projet

Lorsque vous clonez le projet, veuillez effectuer les étapes suivantes :

  1. Remplacez le fichier .env.example par .env.

Configuration de la base de données

Dans le fichier .env, localisez la ligne 11 et configurez les paramètres de votre base de données :

    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=interim_world
    DB_USERNAME=root
    DB_PASSWORD=

Configuration des e-mails

Pour configurer les paramètres d'envoi d'e-mails, suivez les étapes suivantes :

  1. Modifiez la ligne 31 du fichier .env en fonction de votre configuration :
    MAIL_MAILER=smtp
    MAIL_HOST=smtp.gmail.com
    MAIL_PORT=587
    MAIL_USERNAME=votre_email@gmail.com
    MAIL_PASSWORD=votre_mot_de_passe
    MAIL_ENCRYPTION=tls
    MAIL_FROM_ADDRESS=votre_email@gmail.com
    MAIL_FROM_NAME="Interim World"

Utilisation d'un compte Gmail

Si vous souhaitez utiliser un compte Gmail pour envoyer des e-mails, il vous suffit de renseigner votre adresse e-mail et votre mot de passe.

Remarque : Pour les comptes Gmail, vous devez utiliser un mot de passe d'application spécifique plutôt que votre mot de passe habituel.

Voici les étapes pour générer un mot de passe d'application :

  1. Accédez aux paramètres de votre compte Gmail en cliquant sur l'icône de votre compte dans le coin supérieur droit de la page, puis en sélectionnant "Gérer votre compte Google".
  2. Dans le panneau de gauche, cliquez sur l'option "Sécurité".
  3. Activez la vérification à deux facteurs, si ce n'est pas déjà fait.
  4. Faites défiler vers le bas et vous verrez l'option "Mot de passe d'application".
  5. Cliquez sur le bouton "Générer" pour créer un nouveau mot de passe d'application.
  6. Gmail générera un mot de passe d'application unique pour votre application sélectionnée. Copiez ce mot de passe ou notez-le quelque part en lieu sûr.

Exécution du projet

Une fois les étapes de configuration terminées, exécutez la commande suivante pour installer les dépendances :

      composer install

Les identifiants du compte admin doit être défini dans le fichier database/migrations/2023_04_27_142030_create_utilisateurs_table.php à la ligne 29 et 30 (identifiant et mot de passe)

Exécutez les commandes suivantes depuis votre terminal ou votre invite de commande :

    php artisan migrate
    php artisan serve
